# Network Packet Sniffer

This project was completed as part of the CodeAlpha Cybersecurity Internship.

## Description

The program captures live network packets and displays useful information such as source IP address, destination IP address, protocol type, and packet payload.

## Tools Used

- Python
- Scapy
- Visual Studio Code

## What I Learned

- How network packets flow through a system
- Basic understanding of network protocols
- How to capture and analyze packets using Scapy
- Practical application of Python in cybersecurity

## Disclaimer

This project was developed strictly for educational purposes.
