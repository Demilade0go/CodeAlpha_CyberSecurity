# Phishing Awareness Training

This project is part of my CodeAlpha Cyber Security Internship.

The goal of this task is to educate users on phishing attacks, how they work,
and how individuals can protect themselves from becoming victims.

## What the Presentation Covers

- Introduction to phishing attacks
- How to identify phishing emails and fake websites
- Common social engineering techniques used by attackers
- Best practices for avoiding phishing attacks
- Real-world examples and short quiz questions for engagement

## Tools Used

- Microsoft PowerPoint

## Purpose

This project is aimed at improving user awareness and promoting safe online behavior.
